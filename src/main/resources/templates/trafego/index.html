Análise de tráfego

Decodificador didático de pacotes

Cole um hex dump de um quadro/pacote e veja os cabeçalhos Ethernet / IP / TCP / UDP / ICMP decompostos e explicados campo a campo.

info
Sem captura ao vivo. Um navegador/servidor não captura o tráfego da sua máquina — isso exige libpcap + privilégios (Wireshark/tcpdump) num agente local. Aqui a análise é offline: você fornece os bytes (ex.: no Wireshark, botão direito → Copy → as Hex Stream) e o servidor os decodifica.
