DataGrid didático de protocolos: camada, transporte, segurança, função e classificação IGP / EGP para roteamento.
Resumo rápido do módulo de roteamento (PPT): RIP usa Distance Vector (métrica por hops), OSPF/IS-IS usam Link-State (SPF), e BGP usa Path Vector (políticas entre AS). Em laboratório Cisco, destaque para sintaxe com network e uso de wildcard mask no OSPF.
