Framework de Redes — Análise Didática Avançada

Referência técnica: arquitetura, pipeline de análise, fluxos VLSM/WAN, telemetria e observabilidade. Java 25 · Quarkus · monólito modular.

Java 25 Quarkus REST API Docker

lightbulb Visão Geral

Ferramenta didática para estudantes de Redes e Cyber Defense: calcula, valida e explica IPv4/IPv6, CIDR, máscaras, wildcard e VLSM, com catálogos interativos de Portas e Protocolos e um módulo de Resolução de Problemas (VLSM + WAN) que exporta scripts para o Cisco Packet Tracer. Migrado de Python/Flask para Java 25 + Quarkus como um monólito modular (domínios autocontidos, prontos para microserviços).

monitoring Observabilidade

Telemetria server-side com request_id/trace_id por requisição, dashboard em /telemetria e exportação no padrão OpenTelemetry OTLP/JSON.

# baixar os logs em OTLP/JSON

# exportação protegida (chave admin)
