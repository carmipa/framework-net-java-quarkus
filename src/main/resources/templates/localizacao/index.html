info
O que o IP revela — e o que não revela. A geolocalização por IP entrega no máximo a cidade/registro do provedor (ISP), não a residência de uma pessoa. O endereço preciso vem do CEP, que é informado por você — não é derivado do IP. Use este módulo para entender essa diferença, não para "rastrear pessoas".
