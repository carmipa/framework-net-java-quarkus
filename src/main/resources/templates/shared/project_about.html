ℹ️Sobre o Projeto

Este framework foi desenvolvido como ferramenta didática para estudantes de Redes e Cyber Defense, permitindo calcular, validar e explicar conceitos de IPv4, CIDR, máscaras, wildcard, RFC1918, IPv6, comandos Cisco CLI e consultar catálogos técnicos de Portas e Protocolos com DataGrid interativo (filtros, ordenação, paginação e exportação JSON/PDF). Inclui Resolução de Problemas: VLSM para N localidades, topologia WAN (estrela, estrela estendida ou malha), prefixo WAN configurável, EIGRP e OSPF com números da planilha FIAP, exportação de laboratório Packet Tracer (.txt, .zip) e pacote ZIP da turma (um lab por aluno via colagem Excel). O módulo Telemetria oferece dashboard de eventos e console ao vivo.

Competências demonstradas: Java 25 Quarkus Redes TCP/IP IPv4 / IPv6 CIDR / VLSM Portas e Protocolos DataGrid / UX Docker / VPS Telemetria Cyber Defense
